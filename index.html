One-time Google Setup
1

Go to console.cloud.google.com and create a new project

2

Go to APIs & Services → Library, search Tasks API, and click Enable

3

Go to Credentials → Create Credentials → OAuth 2.0 Client ID, choose Web application

4

Under Authorized JavaScript origins add the URL where this app is hosted (e.g. https://yourdomain.com)

5

Copy the Client ID (ends in .apps.googleusercontent.com) and paste below

Google OAuth Client ID
